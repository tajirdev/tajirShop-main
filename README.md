# Tajir Shop

An e-commerce web application that provides a seamless online shopping experience with cart management and checkout functionality.

## Features

### Shopping Cart Management
- **Product Selection**: Users can browse and add products to their shopping cart
- **Cart Recording**: The system automatically tracks all selected products and maintains cart state
- **Cart Page**: Dedicated cart page displaying all selected items with calculated totals

### Checkout Process
- **Cart Review**: Users can review all cart items before proceeding to checkout
- **Price Calculation**: Automatic calculation of total price including all selected products
- **Customer Information Form**: Checkout page includes a form for users to enter their personal and shipping information
- **Payment Methods**: 
  - Cash on Delivery
  - Credit Card Payment
- **Order Placement**: Simple "Place Order" button to complete the purchase

## How It Works

1. **Browse & Add to Cart**: Users select products they want to purchase
2. **View Cart**: All selected items are sent to the cart page with price calculations
3. **Proceed to Checkout**: Users click the checkout button to move to the final step
4. **Fill Information**: Users enter their details and select a payment method
5. **Place Order**: Confirm the order with a single click
